# Setup Guide — 10 minutes to a stunning profile

## Step 1 — Create the special repo
Go to github.com/new and name the repo **exactly** your username: `nagapranathimajji`
Set it to **Public**, initialize with a README. GitHub will show a special banner — that's the one.

## Step 2 — Add the files
Replace the README.md with the one provided.
Create `.github/workflows/snake.yml` with the workflow file provided.

## Step 3 — Enable the snake animation
Go to your repo → **Actions** tab → find "Generate Snake Animation" → click **Run workflow**.
Wait ~1 minute. It creates an `output` branch with your animated SVG.

The snake will auto-update every day at midnight via the cron schedule.

## Step 4 — Pin your best repos
On your GitHub profile page, click **Customize your pins** and select:

- langgraph-mistral-agent  
- Internal-IT-Assistant-RAG-Demo-
